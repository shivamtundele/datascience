Structure of Directory
----------------------------
Logs - Logs for Console Output.
R Visualization - Includes all generated plots.
Rattle - Output folder for Rattle package in R.

File 
------
\Script.R - R-Script for executing complete project.
\widgets.csv - Input dataset.
\final.csv - Final dataset before modelling.
\train.csv - Train dataset.
\test.csv - Test dataset.

Rattle\RandomForestModel.rattle - Model Saved for Random Forest Model.
Rattle\RandomForestScores.csv - Predicted Scores for Random Forest Model
Rattle\RattleLog.txt - Log file generated from Rattle output

Logs\ConsoleOutput - Original.txt - Original Log generated from first run of Complete Project.
